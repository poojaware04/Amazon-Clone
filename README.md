# 🛒 Amazon Clone

### Frontend E-Commerce Website Clone

Amazon Clone is a frontend web development project that recreates the user interface and layout of Amazon's homepage using HTML5 and CSS3. The project focuses on building a visually appealing and responsive e-commerce website while practicing modern web design principles and layout techniques.

The website includes a navigation bar, search functionality UI, hero banner, product showcase sections, shopping cart icon, location section, and a multi-column footer similar to the original Amazon website. This project helped strengthen my understanding of HTML structure, CSS styling, Flexbox layouts, responsive design concepts, and user interface development.
